Summary: Lattitude Vs. X

The purpose of this project was to analyze how weather changes as you get closer to the Equater. In order to retreive data for this analysis we use OpenWeatherAPI to pull weather data for over 500 different cities.

After retreiving our data we used MatPlotlib to plot various weather conditions against the lattitude where they occured. The weather conditions we looked at include the Maximum Temperature, the Humidity, the Cloud Cover, and the Wind Speed. This site is meant to display the data we collected as well as the plots we constructed, along with brief explainations of the trend we found.
